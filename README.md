# Abn

Verify ABN Number

## Usage
Clone the Project

From Project root: 

        bundle
        
        bundle exec bin/abn verify 13488387902  
        bundle exec bin/abn verify <abn>  

## Test
        rspec

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'abn'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install abn
